Detailed Description

template<typename MODEL>
class oops::IncrEnsCtlVec< MODEL >

IncrEnsCtlVec class.

Part of the control vector (acted upon by the localized ensemble B matrix). Consider a general formulation of the state component of the control vector for a particular sub-window \( K \), having two components corresponding to the modelled and ensemble based covariance matrices:

\[ dv^K = \begin{bmatrix} dv^K_{mod} \\ dv^K_{ens} \end{bmatrix} \]

-> \( dv^K \) is implemented by IncrCtlVec

The component corresponding to the localized ensemble B matrix is defined as:

\[ dv^K_{ens} = \begin{bmatrix} dv^K_{ens_{1}} \\ .. \\ dv^K_{ens_{L}} \end{bmatrix} \]

-> \( dv^K_{ens_m} \) is implemented by IncrEnsCtlVec, L is the number of ensemble perturbations used to construct the localized sampled covariance matrix;
