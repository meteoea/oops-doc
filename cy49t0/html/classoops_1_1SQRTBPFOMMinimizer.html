Detailed Description

template<typename MODEL>
class oops::SQRTBPFOMMinimizer< MODEL >

SQRTBPFOM Minimizer.

Preconditioned FOM solver for the square root B-preconditioned formulation of the Hessian

A must be square, symmetric, positive definite.

On entry:

  • A = \( (U^T B^-1 U + U^T H^T R^{-1} U) \)
  • dv = starting point, \( dv_{0} \).
  • rr = \( (-sum dv^{b}_{i} + ) U^T H^T R^{-1} d - A dv0 \)
  • or
  • rr = \( U^T H^T R^{-1} [d + H (x_b - x_fg)] - A dv0 \)

On exit, dv will contain the solution \( U^{-1} dx \) The solution \( dx = U dv \) is recovered in the SQRTMinimizer class The return value is the achieved reduction in preconditioned residual norm.

Iteration will stop if the maximum iteration limit "maxiter" is reached or if the residual norm reduces by a factor of "tolerance".
