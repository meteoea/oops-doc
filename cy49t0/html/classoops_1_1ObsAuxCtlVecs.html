template<typename MODEL>
class oops::ObsAuxCtlVecs< MODEL >

Encapsulates a vector of ObsAuxCtlVec objects.

An object of ObsAuxCtlVecs class is part of ControlVector. The class contains a vector of ObsAuxCtlVec objects. Refer to the documentation of ControlVector and CostJo to understand the structure.
