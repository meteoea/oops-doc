Detailed Description

template<typename MODEL>
class oops::Observations< MODEL >

Observations Class.

Observations class contains observed values or their model equivalents. It contains a vector of ControlObsVector objects and has the same underlying structure as Departures class.

\[ y = \begin{bmatrix} \bar{y}_1 \\ .. \\ \bar{y}_L \end{bmatrix} = \begin{bmatrix} \tilde{y}_1 \\ \tilde{b}_1 \\ .. \\ \tilde{y}_L \\ \tilde{b}_L \end{bmatrix} \]

where \( L \) is the number of ObservationSpace objects held by ObservationSpaces, \( \bar{y}_j \) is implemented by ControlObsVector and has two components: observation \( \tilde{y}_j \) and bias \( \tilde{b}_j \) vectors (each implemented by ObsVector).

Refer to the documentation of CostJo to better understand the structure.
