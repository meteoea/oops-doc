Detailed Description

template<typename MODEL>
class oops::SQRTMinimizer< MODEL >

Abstract base class for square root minimizers.

SQRTMinimizer is the base class for all minimizers that use the square root \( U\) of the \( B = UU^T\) matrix to precondition the variational minimisation problem through the introduction of an auxiliary variable \( dx = U \hat{v} \) or \( dx = U \hat{v} + x_b - x_fg \).

IFS: SQRTPLanczosMinimizer is employed by default, the change of variable \( dx = U \hat{v} + x_b - x_fg \) is default.

Solver (virtual method) - implemented by concrete minimizers.

A.1 Setup minimization specific controls

A.2 Setup the operators B, Jb, UtHtRinvHU

A.3 Instantiate a control increment meant to hold \( H^T R^{-1} d \) or \( H^T R^{-1} [d + H (x_{fg} - x_b) - dv_0] \)

A.4 Instantiate a control vector meant to hold \( U^T H^T R^{-1} d \) or \( H^T R^{-1} [d + H (x_{fg} - x_b) - dv_0] \)

A.5 Instantiate a control vector meant to hold the solution

B.1 Setup the first guess gradient for the classic change of variables \( dx = U dv \)

B.2 Interpolate Jb component of the first guess gradient \( -\sum dv_i \) to the new inner loop resolution

B.3 Compute \( H^T R^{-1} d \)

B.4 Compute \( U^T H^T R^{-1} d \)

B.5 Compute the tolal fist guess gradinet \( -\sum dv_i + U^T H^T R^{-1} d \)

B.6 Set \( J[0] = 0.5 d^T R^{-1} d \)

C.1 Setup the first guess gradient for the change of variables \( dx = U dv + x_b - x_{fg} \)

C.2 Interpolate initial guess control vector to the new inner loop resolution

C.3.a Calculate first guess gradient \( H^T R^{-1} d \)

C.3.b Set \( J[0] = 0.5 d^T R^{-1} d \)

C.3.b Calculate first guess gradient \( H^T R^{-1} [d + H ( dx_{fg} - U dv_0) ] \) and set \( J[0] = 0.5 (d + H ( dx_fg - U dv0))^T R^{-1} (d + H ( dx_fg - U dv0)) \)

C.4 Calculate first guess gradient in control space \( U^T H^T R^{-1} [d + H ( dx_{fg} - U dv_0 ) ] \)

Note that:

\( U^T H^T R^{-1} [d + H ( dx_{fg} - U dv_0 ) ] - J_b dv_0 \)

is equivalent to

\( U^T H^T R^{-1} d + U^T H^T R^{-1} H dx_{fg} - [J_b + U^T H^T R^{-1} H U] dv_0 \)

C.5 Compute \( J_b dv_0 \)

C.6 Initialise \( dv \) to \( dv_0 \)

D.1 Solve the linear system

D.2 Enforce the formulation of the auxiliary observation space component of the control increment

D.3 Calculate the increment \( dx_k = U dv \)

D.4 Calculate quadratic cost function

D.5 Book-keeping for next minimization/(transformation of the increment)

\( dx = U dv \):

Update the Jb component of the first guess gradient

\( dx = U dv + x_b - x_{fg} \) :

Save inital guess

Tranform \( dx_k \) to be defined w.r.t. the first guess

Implements oops::Minimizer< MODEL >.
