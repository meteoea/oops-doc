Detailed Description

template<typename MODEL>
class oops::ControlVector< MODEL >

Control Vector.

The control vector acts as a container for the inputs of the variational data assimilation cost functions in the control variable space. That includes the control vectors at the start the assimilation window or of each sub-window but also additional variables such as model bias, VarBC coefficients, or other control variables for algorithms that use them. The control vector structure is determined by the B matrix.

The control vector \( dv \) has the following structure

\[ dv = \begin{bmatrix} dv_{state4D} \\ dv_{modaux} \\ dv_{obsaux} \end{bmatrix} \]

where 

\f[  dv_{state4D} =  \begin{bmatrix} dv^0 \\
                                     dv^1 \\
                                     ..   \\
                                     dv^N
                      \end{bmatrix}
\f]

and N is the number of sub-windows. For each sub-window:

-> \( dv_{state4D} \) is implemented by IncrCtlVec4D containing a vector of IncrCtlVec objects:

\[ dv^K = \begin{bmatrix} dv^K_{mod} \\ dv^K_{ens} \end{bmatrix} = \begin{bmatrix} dv^K_{mod} \\ \begin{bmatrix} dv^K_{ens_{1}} \\ .. \\ dv^K_{ens_{L}} \end{bmatrix} \end{bmatrix} \]

-> \( dv^K_{mod} \) is implemented by IncrModCtlVec

-> \( dv^K_{ens_m} \) is implemented by IncrEnsCtlVec, L is the number of ensemble perturbations used to construct the localized sampled covariance matrix

\[ dv_{modaux} = \begin{bmatrix} dv_{modaux}^0 \end{bmatrix} \]

-> \( dv_{modaux} \) is implemented by ModelAuxCtlVec

\[ dv_{obsaux} = \begin{bmatrix} dv_{obsaux}^0 \\ dv_{obsaux}^1 \\ .. \\ dv_{obsaux}^K \end{bmatrix} \]

-> \( dv_{obsaux} \) is implemented by ObsAuxCtlVecs which contains a vector of ObsAuxCtlVec objects ( \( dv_{obsaux}^i \)), which in turn contain a map of ObsAuxCtlVecBase objects (e.g \( dv_{VarBC}, dv_{TOVSCV} \)):

\[ dv_{obsaux}^i = \begin{bmatrix} dv_{obsaux,i}^0 \\ .. \\ dv_{obsaux,i}^L \end{bmatrix} \]

-> IFS: K=0, L=1:

\[ dv_{obsaux}= \begin{bmatrix} \begin{bmatrix} dv_{VarBC} \\ dv_{TOVSCV} \end{bmatrix} \end{bmatrix} \]
