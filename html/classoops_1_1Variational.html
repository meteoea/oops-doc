Detailed Description

template<typename MODEL>
class oops::Variational< MODEL >

Variational data assimilation application.

Variational is the top-level driver of variational data assimilation algorithms: 3D-Var, 4D-Var, 4D-Ens-Var and Weak Constraint 4D-Var.

◆ execute()

template<typename MODEL >
int oops::Variational< MODEL >::execute ( const eckit::Configuration &  fullConfig) const
inlinevirtual

Execute a variational application.

  1. Setup the outer-loop geometry
  2. Setup the non-linear model

  3. Setup the cost function; the cost function type (3D-Var, 4D-Var, ..) is selected at run time; the specific cost function object is produced by the cost function factory;

    The background is constructed inside the cost function because its valid time within the assimilation window can be different (3D-Var vs. 4D-Var), it can be 3D or 4D (strong vs weak constraint), etc...

  4. Perform incremental variational assimilation
  5. Skip final trajectory (if the late 4D start mechnism is active)
  6. Obtain the analysis
  7. Save the analysis and compute final diagnostics (evaluate cost function)

Implements oops::Application.
