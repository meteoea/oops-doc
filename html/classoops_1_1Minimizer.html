Detailed Description

template<typename MODEL>
class oops::Minimizer< MODEL >

Abstract base class for minimizers.

Minimizer defines an interface for families of minimizers; it manages the minimization and adjoint/tangent linear model testing. The following families of minimizers are available: DRMinimizer (right B preconditioned minimizers), DualMinimizer (observation space minimizers), PrimalMinimizer (model space minimizers), SQRTMinimizer (square root minimizers), SaddlePointMinimizer (saddle point formulation minimizers), LBMinimizer (left B precondtioned minimizers)

Concrete minimizers are produced by the MinFactory

IFS: SQRTMinimizer is employed by default

Perform model adjoint tests.

Perform the adjoint test of the linear model:

<M dx1, dx2> - <dx1, Mt dx2>)/<M dx1, dx2>

where dx1 and dx2 are increments obtained through randomization

  1. Construct the linear model matrix operator and its adjoint
  2. Instantiate and initialize increments using randomization
  3. Propagate dx1 with the tangent linear model
  4. Propagate backwards dx2 with the adjoint of the linear model
  5. Calculate < M dx1, dx2 >
  6. Calculate < dx1, Mt dx2 >
  7. Print results

template<typename MODEL >
void oops::Minimizer< MODEL >::adjObsTest
private

Perform observation operator adjoint tests.

Perform the adjoint test of the linear observation operator:

(<H dx, dy> - <dx, Ht dy>)/<H dx, dy>

where dx is an increment obtained through randomization and dy is a DualVector obtained by first creating an increment through randomization and then applying a linear observation operator

Note that the \(H = [\overline{H} P] [dx d\beta]\), where \(\overline{H}\) is the linear observation operator and \(P\) is the linear bias operator

A. Perform adjoint test of the composite observation and bias operator \([\overline{H} P] [dx d\beta]\)

Construct linear obs operator matrix and its adjoint

A.1 Instantiate and initialize increments and dual vectors

A.2 Create a new set of increments

A.3 Compute H dx1 and H dx2

A.4 Compute H^T (H dx2)

A.5 Calculate FWD < H dx1, H dx2 >

A.6 Calculate BWD < dx1, H^T (H dx2) >

A.7 Print results

B. Perform adjoint test of the observation operator only \(\overline{H}\). This is achieved by zeroing out the auxiliary observation space component of the control increment

B.1 Reset increments

B.2 Zero out auxiliary obs space component of ControlIncrements

B.3 Compute H dx1 and H dx2

B.4 Copmpute H^T (H dx2)

B.5 Calculate FWD < H dx1, H dx2 >

B.6 Calculate BWD < dx1, H^T (H dx2) >

B.7 Print results

C. Perform adjoint test of the bias operator only \(P\). This is achieved by zeroing out the state increment component of the control increment

C.1 Reset increments

C.2 Reset state increment component of ControlIncrements

C.3 Compute H dx1 and H dx2

C.4 Compute H^T (H dx2)

C.5 Calculate FWD < H dx1, H dx2 >

C.6 Calculate BWD < dx1, Ht (H dx2) >

C.7 Print results

template<typename MODEL >
void oops::Minimizer< MODEL >::tlmTaylorTest
private

Perform the Tayolor test.

TL Taylor test:

||M(x + p dx) - M(x)|| / ||M'(p dx)|| -> 1

where p is a scalar damping factor and dx is initialized using randomization

  1. Construct linear model matrix
  2. Instantiate and initialize the control increment using randomization
  3. Run the non-linear model starting from unperturbed initial conditions (x_b)
  4. Propagate the un-damped perturbation with the tangent linear model: M'(dx)
  5. Loop over increments scaled with increasing damping factor

6.1 Compute ||p M'(dx)||

6.2 Propagate the perturbed state with the non-linear model NL: M(x+pdx)

6.3 Compute ||M(x+pdx) - M(x)||

6.4 Print results
