Detailed Description

template<typename MODEL>
class oops::LinearObsOperator< MODEL >

Encapsulates the linear observation operator.

This class provides the operations associated with the linear observation operator. It wraps the actual linear observation operator which is derived from the linear observation operator base class LinearObsOperBase. The interface for the linear observation operator comprises two levels because we want run time polymorphism. Note that it may be desirable to define a linear observation operator for each individual observation type rather than a single composite operator (which is the case the IFS). LinearObsOperFactory produces concrete objects of LinearObsOperBase children classes.

LinearObsOperators class contains a vector of LinearObsOperator objects. Refer to the documentation of CostJo class to understand the structure of the \( J_o \) term.
