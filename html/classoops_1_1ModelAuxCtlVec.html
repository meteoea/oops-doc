Detailed Description

template<typename MODEL>
class oops::ModelAuxCtlVec< MODEL >

Encapsulates the auxiliary model space component of ControlVector

Part of the control vector corresponding to the ModelAuxIncrement part of ControlIncrement. The structure of ModelAuxCovariances determines the size of ModelAuxCtlVec.
