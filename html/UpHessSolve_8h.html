Detailed Description

Solves the linear system H sol = rhs where H is an n-by-n upper Hessenberg matrix.

Finds the LU factorization of H : H = LU where L is an n-by-n lower unit triangular and U is an n-by-n upper triangular matrix. First solves L y = rhs then solves U sol = y
