Detailed Description

template<typename MODEL>
class oops::ObsAuxCtlVec< MODEL >

Encapsulates auxiliary observation space components of ControlVector.

This class provides operations associated with the auxiliary observation space components (e.g. VarBC, TOVSC) of ControlVector. It contains a vector of ObsAuxCtlVecBase objects, where ObsAuxCtlVecBase is a base class for model specific (or potentially generic) classes.

ObsAuxCtlVecs class wraps around a vector of ObsAuxCtlVec objects and is part of ControlVector. Refer to the documentation of ControlVector and CostJo to understand the structure.
