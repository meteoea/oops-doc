Detailed Description

template<typename MODEL>
class oops::DRMinimizer< MODEL >

DR (Derber and Rosati) Minimizers.

DRMinimizer is the base class for all minimizers that use \( B\) to precondition the variational minimisation problem and use the auxiliary variable \( x = B \hat{x}\) or \( x = B \hat{x} + x_b -x_fg\) and to update it in parallel to \( x\) based on Derber and Rosati, 1989, J. Phys. Oceanog. 1333-1347. \( J_b\) is then computed as \( x^T\hat{x}\) eliminating the need for \( B^{-1}\) or \( B^{1/2}\).
