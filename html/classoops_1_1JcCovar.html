Detailed Description

template<typename MODEL>
class oops::JcCovar< MODEL >

JcCovar class.

This class provides the operations associated with the Jc covariance matrix. It wraps the actual JcCovarBase object which can be a model specific one or a generic one (achieved through polymorphism). The interface for the Jc covariance comprises two levels (JcCovar and JcCovarBase) because we want run time polymorphism.
