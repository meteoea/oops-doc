Detailed Description

template<typename MODEL>
class oops::ObsAuxControlBase< MODEL >

Base class for auxiliary observation space components of ControlVariable. 

Defines interfaces for the auxiliary observation space components of

ControlVariable (e.g. VarBC, TOVSCV). ObsAuxControlFactory is used to produce concrete objects.
