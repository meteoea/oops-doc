Detailed Description

template<typename MODEL>
class oops::JcCovarBase< MODEL >

Base class for encapsulation of the Jc covariances.

Defines the interfaces for Jc covariances. JcCovarFactory is used to produce concrete Jc covariances (model specific or generic).

The public methods (linearize, multiply and inverseMultiply) wrap around the corresponding protected virtual methods (linearize, multiply and inverseMultiply) which are implemented by child model specific or generic classes.
