Detailed Description

template<typename MODEL>
class oops::ObsAuxControls< MODEL >

Encapsulates a vector of ObsAuxControl objects.

An object of ObsAuxControls class is part of ControlVariable. The class contains a vector of ObsAuxControl objects. Refer to the documentation of ControlVariable and CostJo to understand the structure.
