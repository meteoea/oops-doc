Detailed Description

template<typename MODEL>
class oops::ObsVector< MODEL >

Encapsulates the observation vector.

ControlObsVector contains two ObsVector objects to store both observations and biases. Refer to the documentation of ControlObsVector, Observations, Departures and CostJo.
