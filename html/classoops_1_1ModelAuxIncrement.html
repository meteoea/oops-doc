Detailed Description

template<typename MODEL>
class oops::ModelAuxIncrement< MODEL >

Encapsulates the auxiliary model space component of ControlIncrement.

The auxiliary model space component of ControlIncrement is designed as a constant (time step independent) linear model error forcing term that can be jointly estimated by the assimilation algorithm.

\( dx_{k+1} = M_{k+1:k} dx_k + d\eta \), where \( d\eta \) is the model error forcing term increment implemented by ModelAuxIncrement.
