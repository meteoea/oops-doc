Detailed Description

template<typename MODEL>
class oops::ObsAuxCtlVecBase< MODEL >

Base class for auxiliary observation space components of ControlVector.

Defines interfaces for the auxiliary observation space components of ControlVector (e.g. VarBC, TOVSCV). ObsAuxCtlVecFactory is used to produce concrete objects.
