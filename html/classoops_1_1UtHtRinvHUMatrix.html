Detailed Description

template<typename MODEL>
class oops::UtHtRinvHUMatrix< MODEL >

The \( U^T H^T R^{-1} H U\) matrix.

The solvers represent matrices as objects that implement a "multiply" method. This class defines objects that apply a generalized \( U^T H^T R^{-1} H U\) matrix that also includes the equivalent operators for the other terms of the cost function.
