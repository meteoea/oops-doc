Detailed Description

template<typename MODEL>
class oops::IncrementalAssimilation< MODEL >

IncrementalAssimilation implements variational algorithms.

The variational algorithms are implemented using the incremental formulation. IncrementalAssimilation orchestrates the execution of outer loops. Each outer loop consists of three key stages:

  1. Set up the quadratic cost function: compute departures, linearization trajectory, set up linear operators
  2. Minimize the quadratic cost function
  3. Update the first guess

IncrementalAssimilation implements the late 4D start mechanism, orchestrates checkpointing/restart mechanism and warm starts the first guess/minimizer if requested.

IncrementalAssimilation is adapted for single- and multi-executable modes.

IFS: multi-executable mode is default, late 4D start is active by default

Late 4D start mechanism.

Late 4D start mechanism is applied in the last outer loop.

The mechanims consists of the following steps: A.) Record/Save \(x_{fg}^{steps}\) during the first guess trajectory; B.) Propagate the tangent linear model and record \(dx_{fg}^{steps}\); C.) Perform sanity checks on \(dx_{fg}^{steps}\); D.) Obtain 'an' fields: \(x_{an}^{steps} = x_{fg}^{steps} + dx_{fg}^{steps}\); E.) Write out 'an' fields and \(dx_{fg}^{steps}\);

  1. Propagate the increment to the requested times This is achieved by setting up a StateSaver templated on Increment and adding it to the post processor that is attached to the tangent linear model propagation
  2. Retrieve the recorded increments
  3. Perform sanity checks on the retrieved increments based on configuration
  4. Control skipping the final nonlinear trajectory
  5. Write ObsAux, ModAux
  6. Reset linearization, close ODB
  7. Write 4V fields (+0H)
  8. Register the sanity check outcome in the logbook
  9. Retrieve saved FG states if kept in memory
  10. Check if all the sanity tests passed. Skip the final trajecotry if this is the case
  11. Read in written out FG states if they are not available in memory
  12. Release first guess
  13. Compute and write AN fields

13.1 Add increment to the first guess state

13.2 Write AN fields

13.3 Write AN increments

13.4 Release AN fields
