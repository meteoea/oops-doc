Detailed Description

template<typename MODEL>
class oops::ObsErrors< MODEL >

Encapsulates a vector of ObsErrorCovariance objects.

CostJo holds an ObsErrors object. Refer to the documentation of CostJo to understand the structure. Recall we may have multiple ObservationsSpace objects each specific to particular observation types. To that end ObservationSpaces class holds a vector of ObservationSpace objects. ObsErrors class wraps their corresponding ObsErrorCovariance matrices.
