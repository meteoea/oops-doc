Detailed Description

template<typename VECTOR>
class oops::SpectralLMP< VECTOR >

The SPECTRAL preconditioner matrix \( P \) for the system matrix \( AB = (I + Ht Rinv H B) \).

It is defined as \( P_(k+1) = P_k + U (D^{-1} - I_l) X' \) where \( D \) is a l-by-l diagonal matrix whose diagonal entries are \( lambda_i \), \( X = [x_1, x_2, ..., x_l] \) and \( U = [u_1, u_2, ..., u_l] \) are column matrices.

The pairs \( (lambda_i,x_i) \) define the Ritz pairs of the matrix \( PA \) with respect to the subspace \( K(PA,Pr0) \) and \( U = Binv X \). Note that \( r0 \) is the initial residual.

Note that if RitzPrecond_ is active, it applies the RITZ LMP. For details, we refer to S. Gratton, A. Sartenaer and J. Tshimanga, SIAM J. Optim.,21(3),912–935,2011 and S. Gurol, PhD Manuscript, 2013.

The solvers represent matrices as objects that implement a "multiply" method. This class defines objects that apply the preconditioner matrix \( P \).
