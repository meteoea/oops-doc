Detailed Description

template<typename MODEL>
class oops::ModelSpaceCovarianceBase< MODEL >

Abstract base class for model space error covariances.

This class defines interfaces for a family of model space error covariances:

  • ErrorCovariance (modelled B matrix),
  • EnsembleCovariance (sampled B matrix) and
  • HybridCovariance (combining modelled and sampled B matrices).

CovarianceFactory is produces the requested covariances.
