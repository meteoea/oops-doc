Detailed Description

template<typename MODEL>
class oops::ObsAuxCovariances< MODEL >

Encapsulates a vector of ObsAuxCovariance objects.

An object of ObsAuxCovariances class is part of CostJbTotal. The class contains a vector of ObsAuxCovariance objects. Refer to the documentation of ControlVariable, CostJbTotal and CostJo to understand the structure.
