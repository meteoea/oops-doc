Detailed Description

template<typename MODEL>
class oops::CostJb3D< MODEL >

CostJb3D encapsulates the state component of the Jb term of the cost function for a 3 dimensional background (3/4D-Var).

CostJb3D class is not really necessary since it is only a special case of the more general CostJbJq weak constraint term with one sub-window. It is provided for readability.

Consider applying the matrix vector product:

\[ B_{state} dx_{state} = \begin{bmatrix} B_0 \end{bmatrix} \begin{bmatrix} dx_0 \end{bmatrix} \]

\( -> dx_0 \) is implemented by Increment4D containing a single Increment, 

 \f$ -> B_{state} \f$, is implemented through the ModelSpaceCovarianceBase
class; ModelSpaceCovarianceBase is a pure abstract base class for a model
space background error covariance matrix. The following classes inherit
from it:
  - ErrorCovariance - implements a modelled covariance matrix;
  - EnsembleCovariance - implements a localized ensemble covariance matrix;
  - HybridCovariance - implements a hybrid covariance matrix;
