Detailed Description

template<typename MODEL>
class oops::JbMatrix< MODEL >

The \( Jb \) matrix.

The solvers represent matrices as objects that implement a "multiply" method. This class defines objects that apply the \( Jb \) matrix for a general case where U' is not equal to U, where B = UU^t
