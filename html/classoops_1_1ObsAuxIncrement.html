Detailed Description

template<typename MODEL>
class oops::ObsAuxIncrement< MODEL >

Encapsulates auxiliary observation space components of ControlIncrement.

This class provides operations associated with the auxiliary observation space components (e.g. VarBC, TOVSC) of ControlIncrement. It contains a vector of ObsAuxIncrementBase objects, where ObsAuxIncrementBase is a base class for model specific (or potentially generic) classes.

ObsAuxIncrements class wraps around a vector of ObsAuxIncrement objects and is part of ControlIncrement. Refer to the documentation of ControlIncrement and CostJo to understand the structure.
