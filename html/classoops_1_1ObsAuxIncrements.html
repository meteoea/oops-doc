Detailed Description

template<typename MODEL>
class oops::ObsAuxIncrements< MODEL >

Encapsulates a vector of ObsAuxIncrement objects.

An object of ObsAuxIncrements class is part of ControlIncrement. The class contains a vector of ObsAuxIncrement bjects. Refer to the documentation of ControlIncrement and CostJo to understand the structure.
