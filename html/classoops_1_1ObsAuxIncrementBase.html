Detailed Description

template<typename MODEL>
class oops::ObsAuxIncrementBase< MODEL >

Base class for auxiliary observation space components of ControlIncrement.

Defines interfaces for the auxiliary observation space components of ControlIncrement (e.g. VarBC, TOVSCV). ObsAuxIncrementFactory is used to produce concrete objects.
