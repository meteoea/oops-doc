Detailed Description

template<typename FLDS>
class oops::DualConfigPostProcessor< FLDS >

Control model post processing.

This class controls model post processing in the most general sense, ie all diagnostics computations that do not affect the dual config model integration. It just calls all the individual processors one by one.
