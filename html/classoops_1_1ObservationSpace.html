Detailed Description

template<typename MODEL>
class oops::ObservationSpace< MODEL >

Encapsulates the observation space.

ObservationSpace is a bucket for observations; ObservationSpace object can contain observations of specific observation type/types, or it can include observations of all the observation types. This information is agnostic to the OOPS control layer.

ObservationSpace is responsible for performing screening of observations.

ObservationSpaces class contains a vector of ObservationSpace objects. Refer to the documentation of CostJo class to understand the structure of the \( J_o \) term.
