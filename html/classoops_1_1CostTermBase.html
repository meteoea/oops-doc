Detailed Description

template<typename MODEL>
class oops::CostTermBase< MODEL >

Abstract base class for the terms of the cost function (e.g. Jo, Jc).

A vector of CostTermBase objects is stored by the CostFunction allowing for flexibility of the choice of the terms of the cost function.

Methods initialize (returns a post-processor)/finalize are called prior/post the non-linear model trajectory integration.

Methods initializeTraj (returns a post-processor)/finalizeTraj are called prior/post the linearisation trajectory computation by the TrajecotrySaver.

SetupTL/SetupAD set up post-processors-tl/ad attached to the linear/adjoint model integration.
