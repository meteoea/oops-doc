Detailed Description

template<typename MODEL>
class oops::ObsOperator< MODEL >

Encapsulates the observation operator.

This class provides the operations associated with the observation operator. It wraps the actual observation operator which is derived from the operator base class ObsOperatorBase. The interface for the observation operator comprises two levels because we want run time polymorphism. Note that it may be desirable to define an observation operator for each individual observation type rather than a single composite operator (which is the case the IFS). ObsOperatorFactory produces concrete objects of ObsOperatorBase children classes.

ObsOperators class contains a vector of ObsOperator objects. Refer to the documentation of CostJo class to understand the structure of the \( J_o \) term.
