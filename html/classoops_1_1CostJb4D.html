Detailed Description

template<typename MODEL>
class oops::CostJb4D< MODEL >

4D Jb Cost Function

CostJb4D encapsulates the generalized four dimensional Jb term of the 4D-Ens-Var cost function. NOTE: for the square root formulation it is assumed that the localization matrix is constant L_bar = [I I .. I]^T L [I I .. I]
