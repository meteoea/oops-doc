Detailed Description

template<typename INCR>
class oops::PostBaseAD< INCR >

Handles post-processing of model fields related to cost function.

PostBaseAD is the base class for all cost function processors, it is mostly used so that PostProcessorAD can hold a vector of such processors. The difference with PostBase is that the adjoint can modify the increment.
