Detailed Description

template<typename MODEL, typename STATE>
class oops::Observer< MODEL, STATE >

Computes \( {\cal G}(x) \).

Observer is a post-processor that computes \( {\cal G}(x) \). Refer to the documentation of CostJo to understand the structure of \( {\cal G} \).
