Detailed Description

template<typename MODEL>
class oops::Increment4D< MODEL >

Increment4D class.

The state increment contains the increment to the 3D or 4D state part of the control increment.

\( dx_{state4D} \) as implemented by Increment4D implements the state component of a ControlIncrement and contains a vector of Increment objects.

\[ dx_{state4D} = \begin{bmatrix} dx^0 \\ .. \\ dx^K \\ .. \\ dx^M \\ \end{bmatrix} \]

where \( dx^K \) corresponds to the Increment valid for K-th sub-window.
