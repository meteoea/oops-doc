Detailed Description

template<typename MODEL>
class oops::SpectralSqrtLMP< MODEL >

The SPECTRAL preconditioner matrix \( P \) for the system matrix \( A \);.

It is defined as \( P_(k+1) = P_k + U (D^{-1} - I_l) U' \) where \( D \) is a l-by-l diagonal matrix whose diagonal entries are \( lambda_i \), \( U = [u_1, u_2, ..., u_l] \) is a column matrix.

The pairs \( (lambda_i,u_i) \) define the Ritz pairs of the matrix \( PA \) with respect to the subspace \( K(PA,Pr0) \) Note that \( r0 \) is the initial residual.

The solvers represent matrices as objects that implement a "multiply" method. This class defines objects that apply the preconditioner matrix \( P \).
