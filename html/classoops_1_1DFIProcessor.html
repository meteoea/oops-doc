Detailed Description

template<typename MODEL>
class oops::DFIProcessor< MODEL >

Compute time average of states or increments during model run.

Derived classes will compute different types of averages (plain mean, various types of digital filters) by overwriting the weights computation method. This class comes from WeightedMean and allows the coefficients to be different depending on the variable: In ARPEGE DFI, we need coeff=1 at the middle of the assimilation window and 0 elsewhere for GFLs.
