Detailed Description

template<typename MODEL>
class oops::IncrModCtlVec< MODEL >

IncrModCtlVec class.

Part of the control vector (acted upon by the modelled background error covariance matrix). Consider a general formulation of the state component of the control vector for a particular sub-window \( K \), having two components corresponding to the modelled and ensemble based covariance matrices:

\[ dv^K = \begin{bmatrix} dv^K_{mod} \\ dv^K_{ens} \end{bmatrix} \]

-> \( dv^K \) is implemented by IncrCtlVec

-> \( dv^K_{mod} \) is implemented by IncrModCtlVec
