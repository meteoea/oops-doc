Detailed Description

template<typename MODEL>
class oops::ObsAuxCovarianceBase< MODEL >

Base class for covariance of auxiliary observation space components of ControlVariable.

Defines interfaces for the covariances of auxiliary observation space components of ControlVariable (e.g. VarBC, TOVSCV). ObsAuxCovarianceFactory is used to produce concrete objects.
