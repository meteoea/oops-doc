Detailed Description

template<typename MODEL>
class oops::Increment< MODEL >

Encapsulates model increment.

Wrapper around a model specific increment class. Increment is defined as a difference between two states. Some fields that are present in a State may not be present in an Increment.

\( dx_{state4D} \) implemented by Increment4D implements the state component of a ControlIncrement and contains a vector of Increment objects.
