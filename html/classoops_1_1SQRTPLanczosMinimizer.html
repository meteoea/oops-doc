Detailed Description

template<typename MODEL>
class oops::SQRTPLanczosMinimizer< MODEL >

SQRTPLanczosMinimizer.

Preconditioned Lanczos solver for the square root preconditioned formulation of the Hessian

The Hessian must be square, symmetric and positive definite.

On entry:

  • A = \( (U^T B^{-1} U + U^T H^T R^{-1} U) \)
  • dv = starting point, \( dv_0 \).
  • rr = \( (-\sum dv^{b}_{i} + ) U^T H^T R^{-1} d - A dv_0 \)
  • or
  • rr = \( U^T H^T R^{-1} [d + H (x_b - x_{fg})] - A dv_0 \)

On exit, dv will contain the solution such that \( dx = U dv \) or \( dx = U dv + x_{b} - x_{fg} \) The solution is recovered in the SQRTMinimizer class The return value is the achieved reduction in preconditioned residual norm/information content gain.

Iteration will stop if the maximum iteration limit "maxiter" is reached or if the residual norm reduces by a factor of "tolerance"/information content based convergence criterion is reached.
