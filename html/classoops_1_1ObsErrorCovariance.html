Detailed Description

template<typename MODEL>
class oops::ObsErrorCovariance< MODEL >

Encapsulates observation and observation bias error covariance matrices.

This class provides the operations associated with the observation and observation bias error covariance matrices. It wraps the actual observation and observation bias error covariance matrices which can be model specific or generic.

The interface for the observation error comprises two levels (ObsErrorCovariance and ObsErrorBase) because we want run time polymorphism.

The observation and observation bias error covariance matrix has block diagonal structure and is defined as:

\[ \bar{R} \bar{y} = \begin{bmatrix} \tilde{R} && 0 \\ 0 && \tilde{R}_{\beta} \end{bmatrix} \begin{bmatrix} \tilde{y} \\ \tilde{b} \end{bmatrix} \]

where \( \tilde{R} \) is the observation error covariance matrix and \( \tilde{R}_{\beta} \) is the observation bias error covariance matrix. \( \bar{y} \) is implemented by ControlObsVector, which holds two ObsVector components corresponding to observations \( \tilde{y} \) and biases \( \tilde{b} \).

If VarQC is active, it is applied in a square root symmetric form:

\[ \bar{W}^{1/2} \bar{R} \bar{W}^{1/2} \bar{y} = \begin{bmatrix} \tilde{W}^{1/2} && 0 \\ 0 && I \end{bmatrix} \begin{bmatrix} \tilde{R} && 0 \\ 0 && \tilde{R}_{\beta} \end{bmatrix} \begin{bmatrix} \tilde{W}^{1/2} && 0 \\ 0 && I \end{bmatrix} \begin{bmatrix} \tilde{y} \\ \tilde{b} \end{bmatrix} \]

where \( \tilde{W}^{1/2} \) is a diagonal matrix of VarQC weights.

ObsErrors class holds a vector of ObsErrorCovariance objects. Refere to the documentation of CostJo to understand the structure.
