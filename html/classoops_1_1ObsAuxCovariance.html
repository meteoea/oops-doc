Detailed Description

template<typename MODEL>
class oops::ObsAuxCovariance< MODEL >

Encapsulates the covariance matrix of auxiliary observation space components of ControlVariable.

This class provides operations associated with the covariance matrix of auxiliary observation space components (e.g. VarBC, TOVSC) of ControlVariable. It contains a vector of ObsAuxCovarianceBase objects, where ObsAuxCovarianceBase is a base class for model specific (or potentially generic) classes.

ObsAuxCovariances class wraps around a vector of ObsAuxCovariance objects and is part of CostJbTotal. Refer to the documentation of ControlVariable and CostJo to understand the structure.
