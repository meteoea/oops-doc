Detailed Description

template<typename MODEL>
class oops::ModelAuxCovariance< MODEL >

Encapsulates covariance for auxiliary model space component of ControlVariable.

This class provides the operations associated with the covariance matrix of the auxiliary model space component of ControlVariable. It wraps the actual model specific covariance matrix.
