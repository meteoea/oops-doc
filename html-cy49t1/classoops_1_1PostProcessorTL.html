Detailed Description

template<typename INCR>
class oops::PostProcessorTL< INCR >

Control linear model post processing.

This class controls linear model post processing in the most general sense, ie all diagnostics computations that do not affect the linear model integration. It just calls all the individual processors one by one.

Post-processing is organized in three stages/methods: initializeTL (called prior to time-stepping), processTL (called during LinearModel::forecastTL), nd finalize (called once the time-stepping is finished).

Methods initializeTL, processTL and finalizeTL simply call the initializeTL, processTL and finalizeTL methods, respectively, of the enrolled individual processors.

See the documentation of PostBaseTL for the details of implementation of processors and the list of available processors.
