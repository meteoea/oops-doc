Detailed Description

template<typename MODEL>
class oops::ObsAuxControl< MODEL >

Encapsulates auxiliary observation space components of ControlVariable. 

This class provides operations associated with the auxiliary observation

space components (e.g. VarBC, TOVSC) of ControlVariable. It contains a vector of ObsAuxControlBase objects, where ObsAuxControlBase is a base class for model specific (or potentially generic) classes.

ObsAuxControls class wraps around a vector of ObsAuxControl objects and is part of ControlVariable. Refer to the documentation of ControlVariable and CostJo to understand the structure.
